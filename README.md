# vabba# GitHub User Finder

**Автор**: Гайсар Исмагилов

## Описание

GUI‑приложение для поиска пользователей GitHub с возможностью добавления в избранное. Использует официальный GitHub API.

## Функционал

* Поиск пользователей GitHub по имени.
* Отображение результатов в виде списка.
* Добавление пользователей в избранное.
* Сохранение избранных в JSON‑файл.
* Проверка корректности ввода.

## Использование API

Приложение использует GitHub API v3:
* Базовый URL: `https://api.github.com/users/`
* Метод: GET
* Пример запроса: `GET https://api.github.com/users/octocat`
* Ответ возвращается в формате JSON.
